# Weather-app
This is a full-stack weather application built using Flask (Python) for the backend and HTML/CSS/JavaScript for the frontend.
*FEATURES*
	•	Search current weather by city
	•	Get 5-day weather forecast
	•	View air quality information for the city
	•	Responsive and mobile-friendly interface
	•	Info button describing Product Manager Accelerator (PMA)

*HOW IT WORKS*
	•	The frontend sends a request to the Flask backend with the city name.
	•	The backend fetches weather and air quality data from the OpenWeatherMap API.
	•	The results are displayed dynamically on the frontend.

*HOW TO RUN*
	1.	Clone the repository to your local machine.
	2.	Navigate to the backend folder and install the required packages:
    pip install -r requirements.txt
  3.	Run the backend server:
    python 
  4.	Open frontend/index.html in your browser to use the app.

*REQUIREMENTS*
	•	Python 3.x
	•	Flask
	•	Flask-CORS

Note: No database setup is required; the app fetches live data from OpenWeatherMap API.
